# bpgrip

Blender Painting Utility


## Installation

1. Download this add-on in zip
1. Import to Blender
   * `Edit >> Preferences >> Add-ons >> Install` 
1. Enable `bpgrip` 
1. Setup Keymap
   * `Edit >> Preferences >> Keymap >> 3D View >> ... >> Add New`
   * Change `none` to `bpgrip.set_bbmode`
   * Set `Brush Blending Mode` property
